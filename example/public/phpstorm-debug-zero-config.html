Debugging with PHPStorm and XDebug

In case you are always having problems with debugging configuration when using PHPStorm and XDebug (or Zend Debugger), this is the easiest way to instant debug an URL of your web application.

Make sure you did all the steps, then put the 'Start debugger' and 'Stop debugger' into your bookmarks https://www.jetbrains.com/phpstorm/marklets/, set the breakpoints and you are done!

See: https://confluence.jetbrains.com/display/PhpStorm/Zero-configuration+Web+Application+Debugging+with+Xdebug+and+PhpStorm
